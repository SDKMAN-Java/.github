- SDKMAN command not found? Reopen the terminal, reload the shell profile, and verify the install sdkman script completed successfully.  
- Java version not switching? Run sdkman list, choose an installed release, then apply sdkman use java or reinstall with sdkman install java.  
- Build tool missing? Check sdkman candidates for availability, then install sdkman maven or sdkman gradle as needed.  
- Wrong runtime active? Use sdkman current to inspect the session and confirm the expected sdkman jdk or sdkman java version.  
- Cleanup needed? Remove old packages with sdkman uninstall and refresh available data with sdkman update.
